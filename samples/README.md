# Sample Images

These six portraits are bundled with every release so you can try Face Crop Studio's
detection, cropping, and enhancement workflow without supplying your own photos first.

## Origin and licensing

All images in this directory were generated by NVIDIA's StyleGAN2 model and downloaded
from [thispersondoesnotexist.com](https://thispersondoesnotexist.com/). **None of these
faces belong to real people** — every image is a synthetic composite.

Because no real person is depicted:

- There are no rights-of-publicity or model-release concerns.
- There is no copyrightable subject. The StyleGAN2 outputs are treated as public domain
  in practice, and the source site does not impose redistribution restrictions.

You can freely use, modify, or replace these images.

## Replacing the samples

To swap in your own images:

1. Delete the existing `sample_*.jpg` files.
2. Drop in JPEG, PNG, or WebP files of your choosing.
3. Rebuild the release artifacts — every installer (Windows `.msi/.exe/.zip`, macOS `.dmg`,
   Linux `.AppImage/.deb`) bundles the contents of this directory automatically.

## File layout in installed packages

| Platform | Path |
| --- | --- |
| Windows portable zip | `face-crop-studio-windows-x86_64/samples/` |
| Windows installer (NSIS / MSI) | `%ProgramFiles%\Face Crop Studio\samples\` |
| macOS `.app` | `Face Crop Studio.app/Contents/Resources/samples/` |
| Linux `.deb` | `/usr/share/face-crop-studio/samples/` |
| Linux AppImage | `usr/share/face-crop-studio/samples/` (inside the mounted image) |

See [PRESETS.md](PRESETS.md) for the built-in crop and enhancement presets shipped
inside the binary.
